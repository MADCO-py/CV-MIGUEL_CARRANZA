Java

Mi experiencia con Java comenzó en los primeros años de universidad, donde desarrollé una sólida comprensión de la programación orientada a objetos. He trabajado en proyectos académicos que incluyen sistemas de gestión, aplicaciones empresariales y estructuras de datos complejas.

Domino conceptos como herencia, polimorfismo, encapsulación y abstracción. También tengo experiencia con frameworks como Spring y Hibernate para desarrollo de aplicaciones empresariales robustas y escalables.

Python

Python es mi lenguaje favorito para automatización y análisis de datos. Lo he utilizado extensivamente para crear scripts que optimizan procesos, realizar web scraping y desarrollar proyectos de ciencia de datos.

Tengo experiencia con librerías como Pandas, NumPy, Matplotlib y Scikit-learn para análisis de datos y machine learning. También he desarrollado APIs RESTful utilizando Flask y FastAPI, implementando soluciones backend eficientes.

C / C++

Mi conocimiento de C y C++ me ha permitido entender profundamente cómo funcionan las computadoras a bajo nivel. He desarrollado algoritmos eficientes, estructuras de datos avanzadas y aplicaciones que requieren alto rendimiento.

Además, tengo experiencia en programación de microcontroladores con Arduino, donde he creado proyectos de sistemas embebidos e IoT. Este conocimiento me ha dado una perspectiva única sobre la optimización de recursos y el manejo directo de hardware.

Kotlin

Kotlin es mi herramienta principal para el desarrollo de aplicaciones Android modernas. He adoptado Jetpack Compose para crear interfaces de usuario declarativas y reactivas, siguiendo las mejores prácticas de Material Design 3.

Implemento arquitectura MVVM para mantener código limpio y mantenible, y utilizo Coroutines y Flow para manejar operaciones asíncronas de manera eficiente. Mi enfoque está en crear aplicaciones fluidas, intuitivas y con excelente UX.
